Traces are viewable at https://houseofmoran.grafana.net/a/grafana-exploretraces-app/explore — filter by `resource.service.name` to find traces per CLI (e.g. `skeet-prune`, `skeet-feed`, `skeet-live-refine`, `optimise`, `cloudflare-exporter`, `openai-exporter`).
